# Day 7 - Data Visualization with Matplotlib & Seaborn

## Objective
Perform Exploratory Data Analysis (EDA) using Python by creating various visualizations with Matplotlib and Seaborn.

## Dataset
- RetailSales Day7.csv

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Charts Created
1. Sales by Category (Bar Chart)
2. Profit by Region (Bar Chart)
3. Sales Distribution (Histogram)
4. Sales vs Profit (Scatter Plot)
5. Profit Contribution by Category (Pie Chart)
6. Sales Boxplot
7. Correlation Heatmap
8. Monthly Sales Trend (Line Chart)
9. Orders by Region (Count Plot)
10. Subplots (Sales by Category & Profit by Region)

## Key Insights
- Electronics recorded the highest sales.
- Profit varies across different regions.
- Sales data contains a few high-value transactions.
- Sales and Profit show a positive relationship.
- The boxplot identifies outliers in Sales.
- The heatmap highlights correlations among Sales, Quantity, Discount, and Profit.
- Monthly sales fluctuate throughout the year.
- Orders are distributed across all regions.
